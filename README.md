# CLI Математический Калькулятор на Rust 🦀
Учебный консольный калькулятор, написанный на чистом Rust без использования сторонних зависимостей. Программа разбирает сложные математические выражения произвольной длины, учитывает приоритет операторов и поддерживает работу со скобками и отрицательными числами.

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Rust](https://img.shields.io/badge/rust-1.70%2B-orange.svg)](https://www.rust-lang.org)

## ✨ Особенности

- **Алгоритм Дейкстры (Shunting-yard):** Правильная обработка приоритетов операций (`*` и `/` важнее `+` и `-`) и вложенных скобок через систему стеков.
- **Безопасная обработка ошибок:** Идиоматичное использование типов `Result` и `Option` гарантирует, что калькулятор не упадёт («паника») из-за опечатки, пропущенной скобки или деления на ноль.
- **Унарный минус:** Поддержка отрицательных чисел прямо в строке ввода (например, `-5 * (-2)`).

## 🚀 Запуск проекта

1. Склонируйте репозиторий или перейдите в папку с проектом.
   ```bash
   git clone https://github.com/Tuxi4k/simple-calculator.git
   ```
2. Запустите интерактивную консоль калькулятора командой:
   ```bash
   cargo run
   ```

## 💡 Примеры использования

После запуска вводите выражения прямо в консоль после символа `>`:

```text
> 2 + 3 * 4
Результат: 14

> (10 - 2) / 4
Результат: 2

> -2.5 + (3 * -4)
Результат: -14.5

> 5 / 0
Ошибка вычисления: Ошибка: деление на ноль!

> exit
До свидания!
```

## 🛠 Технологический стек и синтаксис Rust

В рамках разработки этого проекта были изучены и применены ключевые концепции языка Rust:

- Мощный сопоставление шаблонов (`match` и макрос `matches!`).
- Управление потоком через итераторы, `peekable()` (подглядывание за символами), циклы `loop` и `while let`.
- Безопасный возврат ошибок и цепочки вызовов с помощью оператора `?` и метода `.map_err()`.
- Расширение возможностей кастомных типов данных через `impl` (реализацию методов) и атрибуты `#[derive(Debug, PartialEq, Clone)]`.
